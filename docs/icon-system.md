# Kairos Patch Icon System

## Direction

Kairos Patch should stay recognizable as part of the wider Kairos family while keeping its own patch/update meaning.

The shared Kairos language is:

- faceted 3D K silhouette
- deep blue/teal left planes, similar to the shaded side of a Minecraft grass block
- bright green right planes, similar to the grass/top face of a Minecraft grass block
- dark negative-space cuts through the center

The Patch-specific layer is:

- crisp desktop-tool geometry
- patch/update meaning through the forward-leaning K shape and broken center cut
- no extra frame, corner scanner, badge, or decorative border

## Assets

- `public/brand/kairos-patch-mark.svg` - primary light mark for app chrome, web favicon source, docs, and light UI.
- `public/brand/kairos-patch-mark-dark.svg` - dark mark for dark backgrounds or release art.
- `public/brand/kairos-patch-full.svg` - horizontal logo lockup for README, release notes, and marketing surfaces.
- `public/brand/kairos-patch-mark-128.png` - raster fallback.
- `public/brand/kairos-patch-mark-256.png` - raster fallback and preview.
- `public/brand/kairos-patch-mark-512.png` - high-resolution raster fallback.
- `public/favicon.svg` - browser favicon.
- `src-tauri/icons/icon.svg` - desktop icon source.
- `src-tauri/icons/icon.ico` - Windows desktop bundle icon.

## Usage

- Use the mark alone inside tight UI spaces.
- Use the full lockup only when there is enough horizontal room.
- Do not use the old purple lightning shape for Kairos Patch surfaces; it does not match the Kairos family.
- Do not return to the box/package variant. The approved direction is the faceted 3D Kairos K mark with Minecraft-grass-block-like color logic.
- Do not add scan-corner border lines to this mark. Kairos Patch should feel like a focused desktop tool, not a console scanner.
- Avoid glow effects around the icon. The mark should read as a product tool icon, not a game power-up.
